About This Expense Tracker Project

This Expense Tracker project was developed as part of a JavaScript Programming course. The goal of the application is to demonstrate DOM manipulation, event handling, input validation, and dynamic content updates.

The app allows users to add expenses, view them instantly, and calculate the total cost in real time. It solves a real-world problem: managing daily expenses in a simple and efficient way.

Skills Used

HTML Structure
CSS Styling
JavaScript DOM
Event Handling

Project Goals

  • Practice JavaScript DOM manipulation
  • Create a real-world expense tracker
  • Improve UI and user experience
  • Build multi-page website navigation
